# 🔧 Exercício 10.46 — Hibbeler

## Lei de Hooke 3D · Deformações num eixo sob torção (cisalhamento puro → deformação a 45°)

</div>

---

## 📋 Enunciado

> O eixo tem **raio de 15 mm** e é feito de **aço-ferramenta L2**.
> Determine as deformações nas direções **x′** e **y′** (a 45° do eixo do tubo),
> se for aplicado um **torque T = 2 kN·m** ao eixo.

<br>

**Dados do material (aço-ferramenta L2, tabela do Hibbeler):**

```text
 E = 200 GPa        G = 75 GPa        (ν ≈ 0,32, pois G = E/[2(1+ν)])
```

<br>

---

## 🧩 Passo 0 · Entender o estado de tensão (a sacada do problema)

Um torque puro num eixo gera, na superfície, **cisalhamento puro**. No elemento
alinhado com o eixo do tubo (**x = eixo do tubo**, **y = circunferencial**):

```text
 σx = 0      σy = 0      σz = 0      τxy = τ   ← SÓ cisalhamento
```

> 🔑 **Por que x′,y′ (e não x,y)?** No cisalhamento puro as deformações **normais
> em x e y são ZERO** (o elemento só "entorta", γxy ≠ 0). Quem estica/encurta são
> as **diagonais a 45°** — exatamente as direções **x′ e y′**. Por isso o problema
> pergunta nelas.

> 📎 Conexão com o resumo (Bloco 2.3): *"cisalhamento puro → ε(a 45°) = τ/(2G) = τ(1+ν)/E"*.

<br>

---

## 🅰️ Passo 1 · Tensão de cisalhamento na superfície (torção)

```text
 c = r = 0,015 m
 J = (π/2)·c⁴ = (π/2)·(0,015)⁴ = 7,952×10⁻⁸ m⁴

 τ = T·c / J = (2000 N·m)(0,015 m) / 7,952×10⁻⁸ m⁴
   = 30 / 7,952×10⁻⁸
   = 377,3 MPa
```

<br>

---

## 🅱️ Passo 2 · x′ e y′ a 45° são as direções PRINCIPAIS

Girando o elemento de cisalhamento puro em 45°, ele vira um elemento só com
**tensões normais** (as principais). No cisalhamento puro:

```text
 σx′ = +τ = +377,3 MPa     (tração ao longo de x′)
 σy′ = −τ = −377,3 MPa     (compressão ao longo de y′)
 σz  =  0
```

> 💡 É o mesmo "círculo de Mohr de tensão" do Cap. 9: cisalhamento puro tem centro
> na origem e principais σ1 = +τ, σ2 = −τ a 45°.

<br>

---

## 🅲 Passo 3 · Lei de Hooke 3D → deformações em x′ e y′

Aplicando Hooke generalizada (Bloco 2 do resumo) com σz = 0:

```text
 εx′ = (1/E)·[ σx′ − ν·σy′ ]
     = (1/E)·[ τ − ν(−τ) ]
     = (τ/E)·(1 + ν)
```

Como **G = E/[2(1+ν)]**, isso é idêntico a **τ/(2G)** — a via mais direta:

```text
 εx′ = τ / (2G) = 377,3×10⁶ / (2 · 75×10⁹)
     = 377,3×10⁶ / 1,5×10¹¹
     = 2,52×10⁻³        (alongamento)

 εy′ = (1/E)·[ σy′ − ν·σx′ ] = −τ/(2G)
     = −2,52×10⁻³       (encurtamento)
```

> ✅ **Conferência:** εx′ e εy′ saem **iguais em módulo e opostos em sinal** — assinatura
> do cisalhamento puro. E note que **εx = εy = 0** (ao longo do eixo/circunferência),
> pois lá o estado é puro cisalhamento (só γxy = τ/G = 5,03×10⁻³).

<br>

---

## 🅳 Passo 4 · Esboço (o elemento a 45°)

```text
        y′ (compressão)
   εy′ = −2,52×10⁻³
            ↕  (encurta)
        ┌────────────┐
        │            │
   ↔    │  elemento  │   ↔   x′ (tração)
        │   a 45°    │       εx′ = +2,52×10⁻³  (alonga)
        └────────────┘
   x′ a 45° do eixo do tubo · só tensões normais (τx′y′ = 0)

   (já no eixo x,y do tubo: εx = εy = 0, só o ângulo reto distorce — γxy = 5,03×10⁻³)
```

<br>

---

## 📊 Resumo

| Grandeza | Valor |
|:---|:---|
| τ (superfície) | **377,3 MPa** |
| εx′ | **+2,52×10⁻³** (tração) |
| εy′ | **−2,52×10⁻³** (compressão) |
| εx = εy (eixo/circunf.) | **0** (cisalhamento puro) |
| γxy | **5,03×10⁻³** |

✅ **Gabarito:** εx′ = +2,52×10⁻³, εy′ = −2,52×10⁻³.

<br>

---

## 🔁 Bônus · Verificação pelo Círculo de Mohr (deformação)

```text
 No eixo x,y (cisalhamento puro):
   εx = εy = 0  →  centro do círculo de deformação em εméd = 0
   γxy/2 = τ/(2G) = 2,52×10⁻³  →  raio R = 2,52×10⁻³

 Ponto x = (εx, γxy/2) = (0, 2,52×10⁻³)
 Girar o elemento 45°  ⇒  girar 2θ = 90° no círculo
   → o ponto cai no eixo horizontal: (εx′, 0) com εx′ = +R = 2,52×10⁻³
   → diametralmente oposto: εy′ = −R = −2,52×10⁻³   ✔️
```

> 🎯 Em deformação a régua é a mesma do Cap. 9 (com γ/2 no eixo vertical): cisalhamento
> puro → círculo centrado na origem; as principais a 45° são ±R = ±τ/(2G).

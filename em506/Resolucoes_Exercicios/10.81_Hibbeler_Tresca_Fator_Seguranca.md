# 🔧 Exercício 10.81 — Hibbeler

## Critério de falha · Tensão de cisalhamento máxima (TRESCA) · Fator de segurança

</div>

---

## 📋 Enunciado

> As **tensões principais no plano** que agem sobre um elemento diferencial são
> mostradas na figura. Se o material for **aço-máquina** com **tensão de escoamento
> σe = 700 MPa**, determine o **fator de segurança** para escoamento, considerando a
> **teoria da tensão de cisalhamento máxima (Tresca)**.

<br>

```text
 Figura (tensões PRINCIPAIS no plano):
       80 MPa → tração      (na horizontal)
       50 MPa → compressão  (na vertical)

 → σ1 = +80 MPa     σ2 = −50 MPa     σ3 = 0   (fora do plano)
 σe = 700 MPa
```

> 🔑 Já são **principais** (a figura mostra só tensões normais, sem cisalhamento nas
> faces). Não precisa de Mohr para achá-las.

<br>

---

## 🧩 Passo 0 · A ideia do critério de Tresca

Como saber se a peça **escoa** (falha)? A teoria de Tresca diz:

```text
 a peça escoa quando o CISALHAMENTO MÁXIMO ABSOLUTO no ponto
 atinge o cisalhamento que causa escoamento no ensaio de tração.
```

Traduzindo em tensões (o τ some e vira diferença de principais):

```text
 |σmáx − σmín| = σe     ← condição de falha (Tresca)
```

> 💡 **De onde vem?** No ensaio de tração simples, no escoamento σ = σe e o
> cisalhamento máximo é σe/2. Como τabs = (σmáx − σmín)/2, igualar os cisalhamentos
> dá (σmáx − σmín)/2 = σe/2 → **σmáx − σmín = σe**. Por isso Tresca usa a **diferença**
> entre a maior e a menor das TRÊS principais (igual ao τ_abs do Mohr 3D, Aula 7).

O **fator de segurança** é o quanto a peça aguenta além do que sofre:

```text
 FS = σe / (σmáx − σmín)
```

<br>

---

## 🅰️ Passo 1 · Listar as TRÊS tensões principais (a pegadinha)

Tresca usa as **três** principais, não só as duas do plano. A terceira é σ3 = 0:

```text
 σ1 = +80     σ2 = −50     σ3 = 0     (MPa)

 Ordenando:   σmáx = +80     σmín = −50
```

> ⚠️ **Atenção ao σ3 = 0.** Aqui as duas do plano têm **sinais opostos** (+80 e −50),
> então a maior e a menor já são elas mesmas — o σ3 = 0 fica **no meio** e não muda nada.
> (Diferente do caso "mesmo sinal", como no vaso esférico/10.23, em que o σ3 = 0 vira o
> extremo e o cisalhamento "sai do plano".)

<br>

---

## 🅱️ Passo 2 · Cisalhamento máximo (a diferença)

```text
 σmáx − σmín = 80 − (−50) = 80 + 50 = 130 MPa
```

> 💡 Equivale a τabs = 130/2 = 65 MPa. Como os sinais são opostos, esse máximo está
> **no próprio plano** (não precisou do σ3 para nada).

<br>

---

## 🅲 Passo 3 · Fator de segurança (Tresca)

```text
 FS = σe / (σmáx − σmín) = 700 / 130 = 5,38
```

> ✅ **Gabarito:  FS = 5,38**
>
> Como FS > 1, a peça **NÃO escoa** — aguenta 5,38× a solicitação atual antes de falhar.

<br>

---

## 📊 Resumo

| Grandeza | Valor |
|:---|:---|
| σ1 / σ2 / σ3 | **+80 / −50 / 0 MPa** |
| σmáx − σmín | **130 MPa** |
| σe (escoamento) | **700 MPa** |
| **FS (Tresca)** | **5,38** |

✅ **Gabarito:** FS = 5,38.

<br>

---

## 🔁 Bônus · Comparação com von Mises (para fixar a diferença)

Se o problema pedisse **von Mises** (energia de distorção), com σ3 = 0:

```text
 σvm = √(σ1² − σ1·σ2 + σ2²) = √(80² − 80·(−50) + (−50)²)
     = √(6400 + 4000 + 2500) = √12 900 = 113,6 MPa

 FS(von Mises) = σe / σvm = 700 / 113,6 = 6,16
```

```text
 Tresca:     FS = 5,38   ← mais CONSERVADOR (FS menor, prevê falha mais cedo)
 von Mises:  FS = 6,16   ← menos conservador
```

> 🎯 **Regra de ouro do capítulo:** Tresca é sempre **mais conservador** que von Mises
> (FS menor). Os dois servem para materiais **dúcteis** (como o aço-máquina daqui).
> Para materiais frágeis usa-se Rankine (tensão normal máxima).

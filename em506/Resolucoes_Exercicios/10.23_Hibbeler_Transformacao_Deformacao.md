# 🔧 Exercício 10.23 — Hibbeler

## Transformação de Deformação · Principais, γ_máx no plano e γ_máx ABSOLUTO

</div>

---

## 📋 Enunciado

> O estado de deformação no ponto **A** do suporte é
> **εx = 300(10⁻⁶)**, **εy = 550(10⁻⁶)** e **γxy = −650(10⁻⁶)**.
> Determine
> **(a)** as deformações principais no plano,
> **(b)** a deformação por cisalhamento máxima no plano e a deformação normal média, e
> **(c)** a deformação por cisalhamento **máxima absoluta**.
> Em cada caso especifique a **orientação** do elemento.

<br>

---

## 🧩 Passo 0 · Estado de deformação dado (montar εx, εy, γxy)

```text
 εx  = +300 ×10⁻⁶     (alonga na horizontal → tração, positivo)
 εy  = +550 ×10⁻⁶     (alonga na vertical → tração, positivo)
 γxy = −650 ×10⁻⁶     (o ângulo reto entre +x e +y ABRE → negativo)
 εz  =    0           (estado PLANO de deformação → ε fora do plano = 0)
```

> 🔑 **A regra do capítulo:** as equações de deformação são **iguais** às de tensão,
> com a troca `σ → ε` e `τ → γ/2`. O cisalhamento entra sempre como **γxy/2**.

<br>

---

## 🅰️ Parte (a) · Deformações principais no plano

**Deformação média e raio (mesma estrutura de σméd e R):**

```text
 εméd      = (εx + εy)/2 = (300 + 550)/2 = 425 ×10⁻⁶
 (εx−εy)/2 = (300 − 550)/2 = −125 ×10⁻⁶
 γxy/2     = −650/2        = −325 ×10⁻⁶

 R = √{ [(εx−εy)/2]² + (γxy/2)² }
   = √{ (−125)² + (−325)² }
   = √{ 15.625 + 105.625 } = √121.250 = 348,2 ×10⁻⁶
```

**Deformações principais:**

```text
 ε1 = εméd + R = 425 + 348,2 =  773 ×10⁻⁶
 ε2 = εméd − R = 425 − 348,2 = 76,8 ×10⁻⁶
```

> ✅ **Conferência (invariante):** ε1 + ε2 = 773 + 76,8 = 849,8 ≈ 850 = εx + εy ✔️

**Orientação dos planos principais:**

```text
 tan(2θp) = γxy / (εx − εy) = −650 / (300 − 550) = −650 / −250 = 2,600
 2θp = 68,96°  ⇒  θp = 34,48°   e   θp − 90° = −55,52°
```

**Qual θp dá ε1?** Substitui θ = 34,48° em εx′:

```text
 εx′ = 425 + (−125)·cos(68,96°) + (−325)·sen(68,96°)
     = 425 − 44,8 − 303,4 = 76,8 ×10⁻⁶ = ε2

 → θp = 34,48°  cai sobre ε2 = 76,8 ×10⁻⁶
 → ε1 = 773 ×10⁻⁶  fica no plano perpendicular:  θp1 = −55,52°
   (55,52° em sentido HORÁRIO a partir de x; nesse plano γ = 0)
```

**Elemento principal (esquema):**

```text
        ε2 = 76,8 ×10⁻⁶
         (alonga pouco)
            ↕
     ┌──────────────┐
     │              │
  ↔  │  principal   │  ↔   ε1 = 773 ×10⁻⁶
     │  (γ = 0)     │      (alonga muito)
     └──────────────┘
   ε1 na direção a 55,52° HORÁRIO a partir de x
```

## 🅱️ Parte (b) · γ_máx no plano + deformação normal média

```text
 γmáx(plano) = 2R = 2 × 348,2 = 696 ×10⁻⁶
 εméd        = 425 ×10⁻⁶     (atua junto, nas duas direções)

 tan(2θs) = −(εx − εy) / γxy = −(−250) / (−650) = 250 / −650 = −0,3846
 2θs = −21,04°  ⇒  θs = −10,52°   e   θs + 90° = 79,48°
```

> ✅ **Conferência:** os planos de γmáx ficam a 45° dos principais →
> θp(ε2) − 45° = 34,48° − 45° = −10,52° ✔️

> ⚠️ A equação devolve **γx′y′/2**; o **γmáx = 2R** já está dobrado (é o valor "cheio").

<br>

## 🧊 Parte (c) · γ_máx ABSOLUTO (a "pegadinha" do capítulo)

Mesmo raciocínio do **τ_abs do Círculo de Mohr 3D (Aula 7)**: ordenar as **três**
deformações principais e tomar a diferença entre a maior e a menor.

```text
 As três principais:  ε1 = 773 ,  ε2 = 76,8 ,  ε3 = εz = 0   (×10⁻⁶)

 Como ε1 e ε2 têm o MESMO sinal (ambas positivas),
 o maior círculo de Mohr NÃO é o do plano — ele envolve o εz = 0.

 εmáx = 773    εmín = 0
 γabs = εmáx − εmín = 773 − 0 = 773 ×10⁻⁶
```

> 🎯 **Por isso γabs (773) > γmáx no plano (696).** Se as principais tivessem
> sinais opostos, o absoluto coincidiria com o do plano. Como são de mesmo sinal,
> entra o **εz = 0** e o cisalhamento máximo "sai do plano".

> 🧊 **Lembrete (Poisson):** numa superfície real esse εz ≠ 0 (estado plano de
> _tensão_, σz=0). Mas neste problema o enunciado dá **estado plano de deformação**,
> então εz = 0 mesmo.

<br>

## 🔁 Bônus · Verificação pelo Círculo de Mohr

```text
 Centro C     = (εméd, 0)     = (425, 0) ×10⁻⁶
 Ponto ref. A = (εx, γxy/2)   = (300, −325) ×10⁻⁶
 Raio R       = √{ (−125)² + (−325)² } = 348,2 ×10⁻⁶

 No plano:   ε1 = C + R = 773      ε2 = C − R = 76,8
             γmáx(plano) = 2R = 696

 Mohr 3D (Aula 7): três pontos no eixo ε → 0 , 76,8 , 773.
 O MAIOR círculo vai de 0 até 773 ⇒ raio = 773/2 = 386,5
 ⇒ γabs = 2 × 386,5 = 773 ×10⁻⁶  ✔️
```

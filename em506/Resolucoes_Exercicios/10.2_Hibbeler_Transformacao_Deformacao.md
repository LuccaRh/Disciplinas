# 🔧 Exercício 10.2 — Hibbeler

## Transformação de Deformação · Estado equivalente para o elemento girado 30°

</div>

---

## 📋 Enunciado

> As componentes do estado plano de deformação no ponto da aba da bequilha são
> **εx = −400(10⁻⁶)**, **εy = 860(10⁻⁶)** e **γxy = 375(10⁻⁶)**.
> Use as equações de transformação da deformação para determinar as deformações
> equivalentes no plano sobre um elemento orientado a um ângulo de
> **θ = 30° em sentido anti-horário** em relação à posição original.
> **Trace um esboço do elemento deformado** devido a essas deformações no plano x–y.

<br>

---

## 🧩 Passo 0 · Estado de deformação dado (montar εx, εy, γxy)

```text
 εx  = −400 ×10⁻⁶     (encurta na horizontal → compressão, negativo)
 εy  = +860 ×10⁻⁶     (alonga na vertical → tração, positivo)
 γxy = +375 ×10⁻⁶     (o ângulo reto entre +x e +y FECHA → positivo)
```

> 🔑 **A regra do capítulo:** as equações de deformação são **iguais** às de tensão,
> com a troca `σ → ε` e `τ → γ/2`. Por isso o cisalhamento sempre entra como **γxy/2**,
> e a equação de cisalhamento devolve **γx′y′/2** (no fim, multiplica por 2).

<br>

---

## 🅰️ Passo 1 · Definir o ângulo de rotação

```text
 θ = +30°   (30° em sentido ANTI-HORÁRIO → positivo)
 2θ = 60°

 cos(2θ) = cos(60°) = +0,5000
 sen(2θ) = sen(60°) = +0,8660
```

<br>

---

## 🅱️ Passo 2 · Aplicar as equações de transformação da deformação

**Constantes úteis:**

```text
 (εx+εy)/2 = (−400 + 860)/2 = 230 ×10⁻⁶
 (εx−εy)/2 = (−400 − 860)/2 = −630 ×10⁻⁶
 γxy/2     = 375/2          = 187,5 ×10⁻⁶
```

**εx′ (deformação normal na direção x′):**

```text
 εx′ = (εx+εy)/2 + [(εx−εy)/2]·cos(2θ) + (γxy/2)·sen(2θ)
     = 230 + (−630)·(0,5000) + (187,5)·(0,8660)
     = 230 − 315 + 162,38
     = 77,4 ×10⁻⁶        (alongamento)
```

**εy′ (deformação normal na direção y′):**

```text
 εy′ = (εx+εy)/2 − [(εx−εy)/2]·cos(2θ) − (γxy/2)·sen(2θ)
     = 230 − (−315) − 162,38
     = 230 + 315 − 162,38
     = 382,6 ×10⁻⁶       (alongamento)
```

> ✅ **Conferência (invariante):** εx′ + εy′ = 77,4 + 382,6 = 460 = εx + εy ✔️

**γx′y′ (deformação angular no plano girado):**

```text
 γx′y′/2 = −[(εx−εy)/2]·sen(2θ) + (γxy/2)·cos(2θ)
         = −(−630)·(0,8660) + (187,5)·(0,5000)
         = 545,60 + 93,75
         = 639,35 ×10⁻⁶

 γx′y′ = 2 × 639,35 = 1.278,7 ×10⁻⁶   ≈ 1.279 ×10⁻⁶
```

> ⚠️ A equação devolve **γx′y′/2**. Para reportar **γx′y′**, multiplique por 2.
> O sinal **positivo** indica que o ângulo reto entre +x′ e +y′ **fecha** (diminui).

<br>

---

## 🅲 Passo 3 · Mostrar o resultado no elemento girado

```text
            εy′ = 382,6 ×10⁻⁶
             (alonga)
                ↕
        ┌────────────────┐
        │   γx′y′ fecha   │
        │   o ângulo      │   εx′ = 77,4 ×10⁻⁶
   ↔    │   (1.279 ×10⁻⁶) │   ↔   (alonga pouco)
        │                │
        └────────────────┘
    eixo x′ a 30° ANTI-HORÁRIO a partir de x
```

> 📐 Interpretação: girado 30° anti-horário, o elemento **alonga pouco em x′**
> (77,4 µε), **alonga bastante em y′** (382,6 µε) e tem o **ângulo reto bem distorcido**
> (γx′y′ = 1.279 µε, fechando).

<br>

---

## 📊 Resumo

| Grandeza | Valor |
|:---|:---|
| εx′ | **77,4 ×10⁻⁶** (alongamento) |
| εy′ | **382,6 ×10⁻⁶** (alongamento) |
| γx′y′ | **1.279 ×10⁻⁶** (ângulo fecha) |
| Orientação | **30° anti-horário** (θ = +30°) |

<br>

---

## 🔁 Bônus · Verificação pelo Círculo de Mohr

```text
 Centro C     = (εméd, 0)     = (230, 0) ×10⁻⁶
 Ponto ref. A = (εx, γxy/2)   = (−400, 187,5) ×10⁻⁶
 Raio R       = √{ [(εx−εy)/2]² + (γxy/2)² }
              = √{ (−630)² + (187,5)² }
              = √{ 396.900 + 35.156 } = √432.056 = 657,3 ×10⁻⁶

 Girar o elemento 30° ANTI-HORÁRIO  ⇒  girar 2θ = 60° ANTI-HORÁRIO no círculo.
 As coordenadas do novo ponto (εx′, γx′y′/2) reproduzem:
   εx′ = 77,4 ×10⁻⁶   e   γx′y′/2 = 639,35 ×10⁻⁶  ⇒  γx′y′ = 1.279 ×10⁻⁶  ✔️
```

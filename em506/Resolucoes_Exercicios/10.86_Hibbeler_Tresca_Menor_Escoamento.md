# 🔧 Exercício 10.86 — Hibbeler

## Critério de falha · TRESCA · Projeto: menor tensão de escoamento (σe) admissível

</div>

---

## 📋 Enunciado

> O **estado de tensão** num ponto crítico da estrutura de um banco de automóvel
> durante uma **colisão** é o da figura. Determine a **menor tensão de escoamento**
> de um aço que possa ser escolhido para o elemento estrutural, com base na
> **teoria da tensão de cisalhamento máxima (Tresca)**.

<br>

```text
 Figura (estado de tensão NÃO-principal → tem cisalhamento):
       σx = 560 MPa   (tração)
       τxy = 175 MPa  (cisalhamento)
       σy = 0
```

> 🔑 **Diferença para o 10.81:** lá as tensões já eram principais. Aqui há **τ na
> figura**, então primeiro preciso achar as principais (Mohr / fórmula do Cap. 9),
> e só depois aplicar Tresca.

<br>

---

## 🧩 Passo 0 · A lógica do "projeto" (engrenagem ao contrário do 10.81)

```text
 10.81 (medir folga):  tinha σe → calculou FS = σe/(σmáx−σmín)
 10.86 (projetar):     quer o MENOR σe → põe FS = 1 (no limite) e isola σe
```

A condição de falha de Tresca é **σmáx − σmín = σe**. O "menor σe que ainda segura"
é exatamente o que iguala essa condição (qualquer aço com σe maior está seguro):

```text
 menor σe = σmáx − σmín       ← o aço precisa ter AO MENOS isso
```

<br>

---

## 🅰️ Passo 1 · Achar as tensões principais (tem τ → usa a fórmula do Cap. 9)

```text
 σméd = (σx + σy)/2 = (560 + 0)/2 = 280 MPa

 R = √{ [(σx − σy)/2]² + τxy² }
   = √{ (280)² + (175)² }
   = √{ 78 400 + 30 625 } = √109 025 = 330,2 MPa
```

```text
 σ1 = σméd + R = 280 + 330,2 = 610,2 MPa
 σ2 = σméd − R = 280 − 330,2 = −50,2 MPa
 σ3 = 0   (fora do plano)
```

> ✅ **Conferência:** σ1 + σ2 = 610,2 − 50,2 = 560 = σx + σy ✔️

<br>

---

## 🅱️ Passo 2 · Ordenar as TRÊS principais e tomar a diferença

```text
 As três:  σ1 = +610,2 ,  σ2 = −50,2 ,  σ3 = 0   (MPa)

 σmáx = +610,2     σmín = −50,2
```

> ⚠️ **Sinais opostos** (+610,2 e −50,2) → a maior e a menor são as duas do plano;
> o σ3 = 0 fica no meio e não influencia. (Mesma situação do 10.81.)

```text
 σmáx − σmín = 610,2 − (−50,2) = 660,4 MPa
```

<br>

---

## 🅲 Passo 3 · Menor tensão de escoamento (Tresca, no limite)

```text
 menor σe = σmáx − σmín = 660,4 MPa
```

> ✅ **Gabarito:  σe(mín) = 660,4 MPa**
>
> Qualquer aço com **σe ≥ 660,4 MPa** resiste sem escoar. Esse é o aço "mais fraco"
> que ainda aguenta a colisão neste ponto.

<br>

---

## 📊 Resumo

| Grandeza | Valor |
|:---|:---|
| σx / τxy | **560 / 175 MPa** |
| σméd / R | **280 / 330,2 MPa** |
| σ1 / σ2 / σ3 | **+610,2 / −50,2 / 0 MPa** |
| σmáx − σmín | **660,4 MPa** |
| **menor σe (Tresca)** | **660,4 MPa** |

✅ **Gabarito:** σe = 660,4 MPa.

<br>

---

## 🔁 Bônus · O que mudaria com von Mises (Problema 10.87)

Se o critério fosse **energia de distorção** (von Mises), com σ3 = 0:

```text
 σvm = √(σ1² − σ1·σ2 + σ2²)
     = √(610,2² − (610,2)(−50,2) + (−50,2)²)
     = √(372 344 + 30 632 + 2 520) = √405 496 ≈ 636,8 MPa

 → menor σe (von Mises) ≈ 636,8 MPa
```

```text
 Tresca:     σe ≥ 660,4 MPa   ← exige aço MAIS forte (conservador)
 von Mises:  σe ≥ 636,8 MPa   ← exige aço um pouco mais fraco
```

> 🎯 Coerente com a regra: **Tresca é mais conservador** → pede mais resistência do
> material. (Esse 636,8 é exatamente o gabarito do 10.87, que resolve este mesmo
> problema por von Mises.)

# 🔧 Exercício 9.15 — Hibbeler

## Transformação de Tensão · Tensões principais e τ_máx no plano

</div>

---

## 📋 Enunciado

> O estado de tensão em um ponto é mostrado no elemento. Determine
> **(a)** as tensões principais e
> **(b)** a tensão de cisalhamento máxima no plano e a tensão normal média.
> **Especifique a orientação do elemento em cada caso.**

<br>

---

## 🧩 Passo 0 · Ler o estado da figura (montar σx, σy, τxy)

A figura mostra **30 MPa na vertical comprimindo** o elemento e **12 MPa de cisalhamento**, sem tensão horizontal:

```text
 σx  =   0 MPa     (não há tensão horizontal)
 σy  = −30 MPa     (30 MPa vertical, em COMPRESSÃO → negativo)
 τxy = +12 MPa     (cisalhamento)
```

<br>

---

## 🅰️ Parte (a) · Tensões principais

**Tensão média e raio:**

```text
 σméd = (σx + σy)/2 = (0 + (−30))/2 = −15 MPa

 R = √{ [(σx − σy)/2]² + τxy² }
   = √{ [(0 − (−30))/2]² + 12² }
   = √{ 15² + 12² } = √{225 + 144} = √369 = 19,21 MPa
```

**Tensões principais:**

```text
 σ1 = σméd + R = −15 + 19,21 =   4,21 MPa
 σ2 = σméd − R = −15 − 19,21 = −34,21 MPa
```

**Orientação dos planos principais:**

```text
 tan(2θp) = τxy / [(σx − σy)/2] = 12 / 15 = 0,8
 2θp = 38,66°  ⇒  θp1 = 19,33°   e   θp2 = 19,33° − 90° = −70,67°
```

**Qual θp dá σ1?** Substitui θp1 = 19,33° em σx′:

```text
 σx′ = −15 + 15·cos(38,66°) + 12·sen(38,66°)
     = −15 + 11,71 + 7,50 = 4,21 MPa = σ1   ✅

 → θp1 = 19,33° (anti-horário)     →  face de σ1 = 4,21 MPa
 → θp2 = −70,67° (= 70,67° horário) →  face de σ2 = −34,21 MPa
   (nessas faces o cisalhamento é ZERO)
```

**Elemento principal (esquema):**

```text
        σ2 = 34,21 MPa
            ↑
     ┌──────────────┐
     │              │
  →  │  principal   │  →   σ1 = 4,21 MPa
     │              │
     └──────────────┘
            ↓
   girado θp1 = 19,33° anti-horário a partir de x
```

<br>

---

## 🅱️ Parte (b) · τ_máx no plano + tensão normal média

```text
 τmáx(plano) = R = 19,21 MPa
 σméd        = −15 MPa     (atua nas DUAS faces — não é zero!)

 tan(2θs) = −[(σx − σy)/2] / τxy = −15 / 12 = −1,25
 2θs = −51,34°  ⇒  θs1 = −25,67°   e   θs2 = −25,67° + 90° = 64,33°
```

> ✅ **Conferência:** os planos de τmáx ficam a 45° dos principais → θp1 − 45° = 19,33° − 45° = −25,67°.

**Elemento de cisalhamento máximo (esquema):**

```text
        σméd = 15 MPa
            ↑
     ┌──────────────┐
     │   →  τmáx     │
     │  19,21 MPa    │
     │   ←          │
     └──────────────┘
            ↓
   girado θs1 = 25,67° HORÁRIO a partir de x (45° do principal)
```

<br>

---

## 📊 Resumo

| Grandeza | Valor |
|:---|:---|
| σ1 | **4,21 MPa** |
| σ2 | **−34,21 MPa** |
| θp1 / θp2 | **19,33° / −70,67°** |
| τ_máx (plano) | **19,21 MPa** |
| σ_méd | **−15 MPa** |
| θs1 / θs2 | **−25,67° / 64,33°** |

<br>

## 🔁 Bônus · Pelo Círculo de Mohr (= Problema 9.65)

```text
 Centro C         = (σméd, 0)    = (−15, 0)
 Ponto ref. A     = (σx, τxy)    = (0, 12)
 Raio R           = 19,21 MPa

 σ1 = C + R = 4,21 MPa     σ2 = C − R = −34,21 MPa
 Ângulo no círculo = 2× ângulo no elemento
```

# 🔧 Exercício 10.92 — Hibbeler

## Critério de falha · TRESCA · Projeto: diâmetro de eixo (torção + flexão + axial combinados)

</div>

---

## 📋 Enunciado

> Numa seção crítica do **eixo de acionamento de aço de um navio**, as cargas internas são:
> **torque T = 3,45 kN·m**, **momento fletor M = 2,25 kN·m** e **propulsão axial P = 12,5 kN**.
> Sendo os limites de escoamento **σe = 700 MPa** (tração) e **τe = 350 MPa** (cisalhamento),
> determine o **diâmetro exigido** para o eixo pela **teoria da tensão de cisalhamento
> máxima (Tresca)**.

<br>

```text
 T = 3,45 kN·m     M = 2,25 kN·m     P = 12,5 kN (axial)
 σe = 700 MPa      τe = 350 MPa  (= σe/2, coerente com Tresca)
 Eixo MACIÇO de diâmetro d (a incógnita)
```

<br>

---

## 🧩 Passo 0 · Montar as TENSÕES no ponto crítico (em função de d)

Cada carga gera uma tensão na superfície do eixo (raio c = d/2):

```text
 Propriedades da seção circular maciça:
   A = π d²/4        I = π d⁴/64        J = π d⁴/32        c = d/2
```

```text
 NORMAL  σ  = axial + flexão (somam no ponto de máxima flexão):
   σ = P/A + M·c/I = 4P/(π d²) + 32M/(π d³)

 CISALHAMENTO τ = torção:
   τ = T·c/J = 16T/(π d³)
```

> 🔑 **A montagem é a chave:** flexão e axial dão **tensão normal** (mesma direção, o
> eixo do navio); torção dá **cisalhamento**. No ponto crítico (fibra externa onde a
> flexão é máxima) a flexão **soma** com a axial. Sobra um estado simples: σ (normal)
> + τ (cisalhamento), igual ao do 10.86 — só que agora σ e τ dependem de **d**.

<br>

---

## 🅰️ Passo 1 · Condição de Tresca em função de σ e τ

Para o estado (σx = σ, σy = 0, τxy = τ), as principais são:

```text
 σ1,2 = σ/2 ± R ,  com  R = √{ (σ/2)² + τ² }
 → σ1 > 0 e σ2 < 0 (sinais opostos) , σ3 = 0 no meio
```

Tresca (no limite, FS = 1): σmáx − σmín = σe. Como σmáx − σmín = σ1 − σ2 = 2R:

```text
 2R = √{ σ² + 4τ² } = σe          ← EQUAÇÃO DE PROJETO (Tresca)
```

> 💡 Equivale a dizer τabs = R = τe = σe/2 = 350 MPa. É a mesma condição, só escrita
> com o cisalhamento. Por isso o enunciado deu τe = 350 = σe/2.

<br>

---

## 🅱️ Passo 2 · Primeira estimativa (desprezando o axial — fórmula fechada)

A propulsão axial (12,5 kN) é pequena. Ignorando-a por ora, σ e τ ficam só com d³:

```text
 σ ≈ 32M/(π d³)      τ = 16T/(π d³)

 σ² + 4τ² = (32/π d³)²·M² + 4·(16/π d³)²·T² = (32/π d³)²·(M² + T²)

 √{σ² + 4τ²} = (32/π d³)·√(M² + T²) = σe
```

Isolando d (fórmula clássica de eixo por Tresca):

```text
 d³ = 32·√(M² + T²) / (π·σe)

 √(M² + T²) = √(2,25² + 3,45²) = √(5,06 + 11,90) = √16,97 = 4,12 kN·m

 d³ = 32·(4,12×10³) / (π·700×10⁶) = 5,99×10⁻⁵ m³
 d  = (5,99×10⁻⁵)^(1/3) = 0,03913 m ≈ 39,1 mm
```

<br>

---

## 🅲 Passo 3 · Refinar incluindo o axial (a equação completa)

Agora com o termo axial 4P/(π d²) de volta, resolve-se:

```text
 √{ [ 4P/(π d²) + 32M/(π d³) ]² + 4·[ 16T/(π d³) ]² } = σe = 700 MPa
```

Resolvendo (numérico, pois mistura d² e d³):

```text
 d ≈ 39,2 mm
```

Conferindo as tensões nesse d:

```text
 σ = 4P/(π d²) + 32M/(π d³) ≈ 10,3 + 379,3 ≈ 389,6 MPa
 τ = 16T/(π d³)             ≈ 290,8 MPa
 2R = √(389,6² + 4·290,8²)  ≈ 700 MPa = σe   ✔️
```

> ✅ **Diâmetro mínimo: d ≈ 39,2 mm.**
> O gabarito do livro traz **d = 39,35 mm** (arredondamento a favor da segurança).
> Na prática, **arredonda-se para cima** → adotar **d = 40 mm**.

<br>

---

## 📊 Resumo

| Grandeza | Valor |
|:---|:---|
| σ (axial + flexão) | **≈ 389,6 MPa** |
| τ (torção) | **≈ 290,8 MPa** |
| σ1 / σ2 | **+544,8 / −155,2 MPa** |
| σmáx − σmín | **= σe = 700 MPa** (no limite) |
| **d mínimo** | **≈ 39,2 mm** (gabarito 39,35 mm) |
| d adotado (arredondar ↑) | **40 mm** |

✅ **Gabarito:** d ≈ 39,35 mm (→ usar 40 mm).

<br>

---

## 🔁 Bônus · A "receita" de eixo sob M + T (memorizar)

```text
 Tresca:     d³ = (32/π σe)·√(M² + T²)            ← usa σe
 von Mises:  d³ = (32/π σe)·√(M² + 0,75·T²)       ← usa σe, pune menos o torque
```

> 🎯 Sem o axial, essas duas fórmulas fechadas resolvem 90% dos problemas de eixo.
> A diferença é o peso do torque: Tresca conta T "cheio"; von Mises conta 0,75·T².
> O axial entra só como correção (termo 4P/π d²), quando existe — como aqui.

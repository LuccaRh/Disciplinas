# 🔧 Exercício 10.96 — Hibbeler

## Critério de falha · TENSÃO NORMAL MÁXIMA (RANKINE) · Material FRÁGIL (concreto)

</div>

---

## 📋 Enunciado

> Um pequeno **cilindro de concreto** de **diâmetro 50 mm** está sujeito a um
> **torque T = 500 N·m** e a uma **força de compressão axial de 2 kN**. Determine se
> ele **falhará** segundo a **teoria da tensão normal máxima (Rankine)**.
> O limite de resistência do concreto é **σr = 28 MPa**.

<br>

```text
 d = 50 mm   →  c = d/2 = 25 mm = 0,025 m
 T = 500 N·m       P = 2 kN  (axial, COMPRESSÃO → sinal negativo)
 σr = 28 MPa       Material FRÁGIL (concreto) → critério de RANKINE
```

> 🔑 **Por que Rankine?** Concreto é **frágil** (quebra de vez, quase não escoa). Para
> frágeis o critério é a **tensão normal máxima**: falha quando a maior tensão normal
> (tração OU compressão) atinge a resistência σr. Aqui a pergunta é "sim/não" (falha?),
> não fator de segurança.

<br>

---

## 🧩 Passo 0 · A ideia de Rankine

```text
 Rankine (tensão normal máxima): compara as principais com σr nas DUAS pontas.

 Falha se:   σmáx ≥ +σr   (rompe por tração)   OU   σmín ≤ −σr   (esmaga por compressão)
 NÃO falha se:   −σr ≤ σmín   E   σmáx ≤ +σr
```

> 💡 Diferente de Tresca/von Mises (que usam diferença ou energia), Rankine olha a
> **própria tensão normal** — porque o frágil rompe quando a tração (ou compressão)
> local fica grande demais, não por cisalhamento.

<br>

---

## 🅰️ Passo 1 · Tensões no ponto crítico (em função das cargas)

Seção circular maciça:

```text
 A = (π/4)·d² = (π/4)·(0,050)² = 1,963×10⁻³ m²  = 1963,5 mm²
 J = (π/32)·d⁴ = (π/32)·(0,050)⁴ = 6,136×10⁻⁷ m⁴
```

```text
 NORMAL (axial, COMPRESSÃO):
   σ = −P/A = −2000 / 1,963×10⁻³ = −1,019×10⁶ Pa = −1,019 MPa

 CISALHAMENTO (torção):
   τ = T·c/J = (500)(0,025) / 6,136×10⁻⁷ = 20,37×10⁶ Pa = 20,37 MPa
```

> ⚠️ O axial é **negativo** (compressão). A torção domina (τ ≈ 20× maior que σ).

<br>

---

## 🅱️ Passo 2 · Tensões principais (estado σ + τ)

```text
 σméd = (σ + 0)/2 = −1,019/2 = −0,509 MPa

 R = √{ (σ/2)² + τ² } = √{ (−0,509)² + (20,37)² }
   = √{ 0,259 + 414,9 } = √415,2 = 20,38 MPa
```

```text
 σ1 = σméd + R = −0,509 + 20,38 = +19,87 MPa     (tração)
 σ2 = σméd − R = −0,509 − 20,38 = −20,89 MPa     (compressão)
 σ3 = 0
```

> 💡 Como há quase só torção (cisalhamento puro + um tiquinho de compressão), as
> principais saem quase **simétricas** (±20), típico de cisalhamento puro.

<br>

---

## 🅲 Passo 3 · Aplicar Rankine (checar as DUAS pontas)

```text
 PONTA DE TRAÇÃO:     σmáx = σ1 = +19,87 MPa   vs   +σr = +28 MPa
     19,87 < 28   → OK, não rompe por tração ✔️

 PONTA DE COMPRESSÃO: σmín = σ2 = −20,89 MPa   vs   −σr = −28 MPa
     −20,89 > −28  (|20,89| < 28)  → OK, não esmaga por compressão ✔️
```

> ✅ **As duas pontas passam → o cilindro NÃO falha.**

<br>

---

## 📊 Resumo

| Grandeza | Valor |
|:---|:---|
| σ (axial, compressão) | **−1,019 MPa** |
| τ (torção) | **20,37 MPa** |
| σ1 (tração) | **+19,87 MPa** |
| σ2 (compressão) | **−20,89 MPa** |
| σr (resistência) | **28 MPa** |
| Tração: σ1 < σr ? | **19,87 < 28 ✔️** |
| Compressão: \|σ2\| < σr ? | **20,89 < 28 ✔️** |

✅ **Gabarito:** **NÃO falha** (σ1 = 19,87 MPa < 28 MPa, e \|σ2\| = 20,89 < 28).

<br>

---

## 🔁 Bônus · Por que Rankine (e não Tresca/von Mises) aqui

```text
 FRÁGIL (concreto, ferro fundido, vidro):
   rompe na maior TENSÃO NORMAL  → Rankine: σmáx ≤ σr (checa tração E compressão)

 DÚCTIL (aço, alumínio):
   escoa no CISALHAMENTO/energia → Tresca (σmáx−σmín) ou von Mises (σvm)
```

> 🎯 Se por engano usássemos **Tresca** aqui: σmáx − σmín = 19,87 − (−20,89) = 40,76 MPa
> > 28 → "falharia". **Resultado errado**, porque o concreto não falha por cisalhamento.
> Escolher o critério certo (pelo **tipo de material**) é o passo decisivo — o mesmo
> estado de tensão dá veredito oposto conforme a teoria. Para frágil, **Rankine**.

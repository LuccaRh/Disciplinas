# 🔧 Exercício 9.90 — Hibbeler

## Transformação de Tensão · Tensões principais e cisalhamento máximo ABSOLUTO (estado 3D)

</div>

---

## 📋 Enunciado

> **A tensão em um ponto é mostrada no elemento.** Determine as **tensões
> principais** e a **tensão de cisalhamento máxima absoluta**.

<br>

---

## 🧩 Passo 0 · Ler o estado da figura (montar o tensor)

O elemento é **tridimensional** (eixos x, y, z). Lendo as setas:

```text
 σx  = −100 MPa   (seta de 100 MPa aponta PARA DENTRO da face x → compressão)
 σy  = +90  MPa   (seta de 90 MPa aponta PARA FORA da face y  → tração)
 σz  = −80  MPa   (seta de 80 MPa aponta PARA BAIXO na face z  → compressão)
 τyz = +40  MPa   (cisalhamento no plano y–z; seta vertical na face y + complementar no topo)
 τxy = τzx = 0
```

Tensor de tensões:

```text
        ┌  −100     0      0  ┐
  σ  =  │    0    +90    +40  │   (MPa)
        └    0    +40    −80  ┘
```

> 🔎 **Observação-chave:** a face **x não tem cisalhamento** (τxy = τzx = 0).
> Logo **σx = −100 MPa já é uma tensão principal**, e o eixo **x é uma direção
> principal**. O problema 3D se reduz a uma análise **plana no plano y–z**.

<br>

---

## 🅰️ Passo 1 · Análise no plano y–z (estado plano de tensão)

No plano y–z temos σy = +90, σz = −80, τyz = 40 MPa.

**Tensão média e raio de Mohr (nesse plano):**

```text
 σméd = (σy + σz)/2 = (90 + (−80))/2 = 5 MPa

 R = √{ [(σy − σz)/2]² + τyz² }
   = √{ [(90 − (−80))/2]² + 40² }
   = √{ 85² + 40² }
   = √{ 7225 + 1600 }
   = √8825
   = 93,94 MPa
```

**Tensões principais no plano y–z:**

```text
 σa = σméd + R = 5 + 93,94 =  98,94 MPa
 σb = σméd − R = 5 − 93,94 = −88,94 MPa
```

**Orientação do plano principal (no plano y–z):**

```text
 tan(2θp) = τyz / [(σy − σz)/2] = 40 / 85 = 0,4706
 2θp = 25,2°   →   θp ≈ 12,6°
```

<br>

---

## 🅱️ Passo 2 · Reunir as TRÊS tensões principais

Juntando a principal "de fora do plano" (σx) com as duas do plano y–z:

```text
 σx = −100,00 MPa   (já era principal)
 σa =  +98,94 MPa
 σb =  −88,94 MPa
```

Ordenando do maior para o menor (convenção σmáx ≥ σint ≥ σmín):

```text
 σmáx  =  +98,94 MPa
 σint  =  −88,94 MPa
 σmín  = −100,00 MPa
```

> ✅ **Conferência (traço invariante):**
> σx + σy + σz = −100 + 90 − 80 = −90 MPa
> σmáx + σint + σmín = 98,94 − 88,94 − 100,00 = −90 MPa ✔️

<br>

---

## 🅲 Passo 3 · Tensão de cisalhamento máxima ABSOLUTA

A máxima absoluta vem do **maior dos três círculos de Mohr**, ou seja, do par
(σmáx, σmín):

```text
 τabs_máx = (σmáx − σmín)/2 = (98,94 − (−100,00))/2 = 198,94/2 = 99,47 MPa
```

Tensão normal no plano desse cisalhamento (centro do maior círculo):

```text
 σméd = (σmáx + σmín)/2 = (98,94 − 100,00)/2 = −0,53 MPa
```

<br>

---

## 📐 Os três Círculos de Mohr (visão 3D)

```text
        τ (MPa)
        │
  99,47 ┤              ╭───────────────────╮      ← MAIOR círculo
        │           ╭──┤         ●         ├──╮      (σmáx, σmín) → τabs
        │         ╭─┤  │   ╭───╮ │ ╭─────╮ │  ├─╮
      0 ┼────●────┼──┼──┼──┤   ├─┼─┤     ├─┼──┼──●──── σ (MPa)
        │  −100  −88,94    −0,53        +98,94
        │  σmín   σint                   σmáx
        └──────────────────────────────────────

 Círculo 1 (maior): entre σmáx=98,94 e σmín=−100   → R = 99,47  (= τabs_máx)
 Círculo 2:         entre σmáx=98,94 e σint=−88,94 → R = 93,94
 Círculo 3:         entre σint=−88,94 e σmín=−100  → R =  5,53
```

> A **tensão de cisalhamento máxima absoluta** é sempre o raio do **maior**
> círculo — controlado pela maior e pela menor tensão principal.

<br>

---

## 📊 Resumo

| Grandeza | Valor |
|:---|:---|
| σmáx | **+98,94 MPa** |
| σint | **−88,94 MPa** |
| σmín | **−100,00 MPa** |
| τmáx absoluta | **99,47 MPa** |
| σméd (no plano de τabs) | −0,53 MPa |
| Orientação no plano y–z | θp ≈ 12,6° |

<br>

> 🎯 **Bate com o gabarito do livro:**
> σmáx = 98,94 · σint = −88,94 · σmín = −100,00 · τabs = 99,47 MPa ✔️

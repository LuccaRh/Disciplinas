# 🔧 Exercício 9.68 — Hibbeler

## Transformação de Tensão · Estado equivalente para o elemento girado 30°

</div>

---

## 📋 Enunciado

> **Determine o estado de tensão equivalente** se um elemento estiver
> **orientado a 30° em sentido horário** em relação ao elemento mostrado.

<br>

---

## 🧩 Passo 0 · Ler o estado da figura (montar σx, σy, τxy)

A figura mostra **350 MPa na horizontal tracionando**, **230 MPa na vertical
tracionando** e **480 MPa de cisalhamento**:

```text
 σx  = +350 MPa     (tração horizontal → positivo)
 σy  = +230 MPa     (tração vertical → positivo)
 τxy = −480 MPa     (cisalhamento)
```

**Sinal do cisalhamento (convenção de Hibbeler):** τxy é positivo quando, na
face **+x**, a seta aponta para **+y**. Na figura, na face direita (+x) a seta
de cisalhamento aponta para **baixo (−y)** → portanto **τxy = −480 MPa**.

```text
 face +x (direita)  → cisalhamento ↓ (−y)   ✓ negativo
 face +y (topo)     → cisalhamento ← (−x)   ✓ negativo
 face −x (esquerda) → cisalhamento ↑ (+y)
 face −y (base)     → cisalhamento → (+x)
```

<br>

---

## 🅰️ Passo 1 · Definir o ângulo de rotação

```text
 θ = −30°   (30° em sentido HORÁRIO → negativo)
 2θ = −60°

 cos(2θ) = cos(−60°) = +0,5000
 sen(2θ) = sen(−60°) = −0,8660
```

<br>

---

## 🅱️ Passo 2 · Aplicar as equações de transformação de tensão

**Tensão média e semidiferença (constantes úteis):**

```text
 σméd      = (σx + σy)/2 = (350 + 230)/2 = 290 MPa
 (σx−σy)/2 = (350 − 230)/2 = 60 MPa
```

**σx′ (face normal a x′):**

```text
 σx′ = (σx+σy)/2 + [(σx−σy)/2]·cos(2θ) + τxy·sen(2θ)
     = 290 + 60·(0,5000) + (−480)·(−0,8660)
     = 290 + 30 + 415,69
     = 735,69 MPa   ≈ 736 MPa   (tração)
```

**σy′ (face normal a y′):**

```text
 σy′ = (σx+σy)/2 − [(σx−σy)/2]·cos(2θ) − τxy·sen(2θ)
     = 290 − 30 − 415,69
     = −155,69 MPa   ≈ −156 MPa   (compressão)
```

> ✅ **Conferência:** σx′ + σy′ = 735,69 + (−155,69) = 580 MPa = σx + σy ✔️

**τx′y′ (cisalhamento no plano girado):**

```text
 τx′y′ = −[(σx−σy)/2]·sen(2θ) + τxy·cos(2θ)
       = −60·(−0,8660) + (−480)·(0,5000)
       = 51,96 − 240
       = −188,04 MPa   ≈ −188 MPa
```

> O sinal **negativo** de τx′y′ indica que, na face **+x′**, o cisalhamento
> aponta para **−y′** (sentido oposto ao positivo da convenção).

<br>

---

## 🅲 Passo 3 · Mostrar o resultado no elemento girado

```text
        σy′ = 155,69 MPa
       (compressão)
            │
            ▼
     ┌──────────────┐
     │      ←        │   τx′y′ = 188,04 MPa
 ←   │   girado     │  →   σx′ = 735,69 MPa
     │   30° horário │      (tração)
     │       →       │
     └──────────────┘
            ▲
   eixo x′ a 30° HORÁRIO a partir de x
```

<br>

---

## 📊 Resumo

| Grandeza | Valor |
|:---|:---|
| σx′ | **735,69 MPa** (≈ 736 MPa, tração) |
| σy′ | **−155,69 MPa** (≈ −156 MPa, compressão) |
| τx′y′ | **−188,04 MPa** (≈ −188 MPa) |
| Orientação | **30° horário** (θ = −30°) |

<br>

---

## 🔁 Bônus · Verificação pelo Círculo de Mohr

```text
 Centro C     = (σméd, 0)   = (290, 0)
 Ponto ref. A = (σx, τxy)   = (350, −480)
 Raio R       = √{ [(σx−σy)/2]² + τxy² }
              = √{ 60² + 480² } = √234000 = 483,74 MPa

 Girar o elemento 30° HORÁRIO  ⇒  girar 2θ = 60° HORÁRIO no círculo.
 As coordenadas do novo ponto (σx′, τx′y′) reproduzem:
   σx′ = 735,69 MPa   e   τx′y′ = −188,04 MPa   ✔️
```
